Why frontend, backend, and database are deployed separately?
Frontend backend and database are deployed separately because they have different responsibilities and resource needs and separating them improves scalability security and maintenance

Why GitHub Pages cannot host APIs
GitHub Pages cannot host APIs because it only serves static files and does not support server side code execution

How your Vue app communicates with the API
The Vue app communicates with the API by sending HTTP requests to the deployed backend endpoint and receiving JSON responses

How the API communicates with the database
The API communicates with the database by opening a secure connection running SQL queries and returning the results to the server

What problems you encountered during deployment
During deployment I encountered issues such as network errors incorrect environment variables and CORS restrictions

How you fixed those problems
I fixed these problems by checking configuration files updating API URLs setting proper environment variables and enabling CORS on the backend
